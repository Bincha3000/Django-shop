## Requirements:
* Python>=3.5
* Django==2.2.2
* django-debug-toolbar==2.0
* Pillow==6.0.0


## Run project:

```
git clone git@github.com:gladunvv/django-shop.git
pip install -r requirements.txt
pyhon demo_shop/manage.py runserver
```
